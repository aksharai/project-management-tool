# Project Management Tool (PMT)

A Project Management Tool built using React.js and CSS that helps teams manage tasks efficiently through task assignment, Kanban workflow tracking, team collaboration, and analytics reporting.

## Features

- Task assignment and tracking
- Kanban board workflow (Todo, In Progress, Done)
- Team collaboration with comments
- Project analytics dashboard
- LocalStorage support
- Responsive and clean UI

## Technologies Used

- React.js
- CSS
- JavaScript
- LocalStorage

## Project Structure

src/
- App.js
- App.css
- index.js
- index.css

public/
- package.json

## How to Run

1. Install dependencies
npm install

2. Start the project
npm start

## Output

The application allows users to:
- Add and assign tasks
- Move tasks across workflow stages
- Add comments for collaboration
- Track project progress with analytics
